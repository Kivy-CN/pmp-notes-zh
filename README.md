## 下载

### Docker

```
docker pull apachecn0/pmp-notes-zh
docker run -tid -p <port>:80 apachecn0/pmp-notes-zh
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g pmp-notes-zh
pmp-notes-zh <port>
# 访问 http://localhost:{port} 查看文档
```
